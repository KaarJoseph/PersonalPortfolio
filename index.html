Proyectos

EmotionLens: Sistema de Reconocimiento de Emociones

Una aplicación móvil desarrollada en Android Studio con Java, que utiliza algoritmos de visión por computador para el reconocimiento facial. Emplea cascadas de Haar para detectar rostro, ojos, nariz y boca. Con estos puntos, calcula descriptores (como HOG o similares), los envía a un servidor web implementado en Flask, donde un modelo SVM predice la emoción del usuario. Además, la aplicación aplica un filtro interactivo (como unas gafas virtuales) para mejorar la experiencia visual y educativa. Este sistema es ideal para proyectos educativos y de interacción social.

AMIRA: Conectando con tu Bienestar

AMIRA es un asistente emocional diseñado para interactuar con estudiantes mediante un chat, recopilando información clave de las conversaciones. Utiliza técnicas avanzadas de análisis de texto para identificar patrones relacionados con la ansiedad, depresión y estrés. Con base en estos datos, AMIRA genera un prediagnóstico emocional, ofreciendo una herramienta innovadora para promover el bienestar y facilitar intervenciones oportunas en el ámbito educativo.
